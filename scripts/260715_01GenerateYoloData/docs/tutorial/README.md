# 260715_01GenerateYoloData 学习教程

这里是项目的中文学习入口。教程基于 2026-07-21 工作区中的当前代码编写，讲解对象是当前可配置多类别版本，而不是 Git 初始提交中的旧版固定 <code>tooth</code> 转换逻辑。

## 推荐阅读顺序

1. 从头阅读 [完整学习笔记](./260715_01GenerateYoloData_完整学习笔记.md) 的第 1～7 章，建立数据流和运行方法。
2. 跟随第 8 章手算一次边界框，真正理解 YOLO 坐标。
3. 对照 <code>convert.py</code> 阅读第 9～12 章，理解源码和并发设计。
4. 运行第 13 章的测试，再用第 14～17 章排错和进行二次开发。

## 学完后应能做到

- 解释 RGB、二维语义 ID 数组、semantic mapping 和 YOLO 类别配置之间的关系。
- 独立准备输入并运行转换命令。
- 手工验证任意一行 YOLO 标注是否正确。
- 顺着主程序和单帧处理函数定位问题。
- 正确使用断点续跑、覆盖、错误日志和并发参数。
- 知道当前项目的能力边界，并能规划连通域拆框、训练集组织等扩展。
